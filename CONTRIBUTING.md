# Contribuire a Orfelo Assicurazioni

Grazie per aver considerato di contribuire a Orfelo Assicurazioni! 
Qui di seguito sono riportate alcune linee guida su come puoi contribuire al progetto.

## Segnalazione di Problemi (Issues)
Se riscontri un problema o un bug nell'applicazione, ti invitiamo a segnalarlo aprendo una nuova issue nel repository su GitHub. 
Assicurati di includere una descrizione dettagliata del problema, insieme a eventuali passaggi per riprodurlo.

## Proposte di Miglioramento
Se hai idee su come migliorare l'applicazione o aggiungere nuove funzionalità, puoi proporle aprendo una nuova issue nel repository. 
Assicurati di descrivere chiaramente la tua proposta e fornire eventuali dettagli o mockup che la supportino.

## Contributi al Codice
Se desideri contribuire al codice sorgente di Orfelo Assicurazioni, puoi farlo aprendo una pull request nel repository. 
Assicurati di seguire le linee guida di stile del codice e di documentare adeguatamente le tue modifiche.

## Linee Guida per le Pull Request
- Assicurati che la tua pull request risolva un problema o aggiunga una funzionalità significativa.
- Segui le convenzioni di denominazione e di stile del codice.
- Documenta adeguatamente le tue modifiche, compresi eventuali cambiamenti nelle dipendenze o nelle configurazioni.
- Se la tua pull request riguarda una nuova funzionalità, assicurati di includere test adeguati per verificare il suo corretto funzionamento.
